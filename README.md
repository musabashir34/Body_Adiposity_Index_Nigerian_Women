# Body_Adiposity_Index_Nigerian_Women
This is the data (clean_data.csv), R script for generating tables and figures (Analysis_Script.R) and Rmarkdown file (Article.Rmd) for the research article titled: "Body Adiposity Index Outperforms Central Obesity Measures in Predicting HbA1c Levels among Hausa Women across Menopausal Status" Submitted to the Maturitas Journal
The Analysis_Script.R file was already sourced in the Article.Rmd. So you can knit the Article.Rmd file to generate the complete article
