# Body_Adiposity_Index_Nigerian_Women
This is the data (clean_data.csv), R script for generating tables and figures (Analysis_Script.R) and Rmarkdown file (Article.Rmd) for the research article titled: "Differential Predictive Value of Adiposity Indices for Glycemic Dysregulation Across the Menopausal Transition in Hausa Women" Submitted to Archives of Gerontology and Geriatrics Journal
The Analysis_Script.R file was already sourced in the Article.Rmd. So you can knit the Article.Rmd file to generate the complete article
