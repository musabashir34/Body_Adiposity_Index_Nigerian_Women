# Body_Adiposity_Index_Nigerian_Women
This is the data (clean_data.csv), R script for generating tables and figures (Analysis_Script.R) and Rmarkdown file (Article.Rmd) for the research article titled: "Body Adiposity Index as a Low-Cost Screening Marker for Glycemic Risk in Hausa Women: A Comparison Across the Menopausal Transition" Submitted to Primary Care Diabetes Journal
The Analysis_Script.R file was already sourced in the Article.Rmd. So you can knit the Article.Rmd file to generate the complete article
